# 题目描述

```
将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/merge-two-sorted-lists
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```

# 解题思路
* 指定一个头节点和头指针
* 开始遍历,每次遍历,当前列表节点都移动一次
* 根据大小,将节点依次放置在头节点的位置上,每次放置完,头指针移动一次
