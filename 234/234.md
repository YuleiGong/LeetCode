# 题目描述 

```
请判断一个链表是否为回文链表。
示例 1:
输入: 1->2
输出: false
示例 2:

输入: 1->2->2->1
输出: true
进阶：
你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题？
```

# 解题思路
* 回文:翻转之后,和之前是一致的
* 遍历一次链表,使用stack存储数据
* 存储完成后，再次遍历链表，并从stask中pop数据。如果数据和链表不一致,则不是回文

# 最优思路
* 遍历链表,找到中间节点。
* 根据中间节点，翻转链表
* 对比翻转链表和老链表的值
